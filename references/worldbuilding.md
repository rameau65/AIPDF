# Worldbuilding System

## 세계관 원칙

세계는 장식이 아니라 캐릭터의 감정과 행동을 증폭하는 장치다.

## 기본 장소

- 감정정원: 감정이 식물처럼 자라는 곳
- 상상호수: 기억과 상상이 비치는 곳
- 장난감언덕: 버려진 물건이 새 역할을 얻는 곳
- 꿈시장: 감정과 기억을 교환하는 야시장
- 숨은문 골목: 현실과 세계관을 잇는 입구
- 구름섬: 고요함과 수면을 담당하는 공간

## 장소 설계 항목

- 핵심 감정
- 대표 색
- 대표 소리
- 이동 방식
- 캐릭터 행동
- 에피소드 소재
- 제품화 요소
- 전시 체험 요소
- 게임 플레이 요소

## 스토리 루프

Reality → Curiosity → Hidden Door → Discovery → Emotional Conflict → Playful Resolution → Return

## 캐릭터 관계도

관계는 '친구/적'보다 다음 축으로 정의한다.

- 감정을 드러내게 하는가
- 감정을 숨기게 하는가
- 행동을 촉진하는가
- 성장을 방해하는가
- 세계의 규칙을 설명하는가
