## 파트너 제안서 필수 항목

- IP 한 문장 정의
- 캐릭터·세계관 차별성
- 핵심 타깃
- 대표 비주얼
- 상품 확장성
- 콘텐츠 확장성
- 채널 전략
- 매출 구조
- 3~5년 로드맵
- 요청 투자 또는 파트너십 범위

## 수익 모델

- 직접 판매
- 라이선스 로열티
- 콘텐츠 판권
- 플랫폼 수익
- 브랜드 협업
- 전시 티켓
- 공간 F&B
- 디지털 아이템
- 게임 인앱 결제

## 숫자 표기 규칙

- 확정 수치와 가정치를 분리
- 가정치는 Assumption 라벨 표시
- 시장 규모는 출처와 기준연도 표시
- 매출 전망은 Base / Upside / Downside 시나리오 제시
- 로열티율과 제조원가는 범위로 제시 가능
