# SINDAENG IP Profile

## 기본 정보

- 국문명: 신댕이
- 영문 표기: SINDAENG
- 분류: 감정 기반 캐릭터 IP
- 핵심 장르: 판타지, 일상, 성장, 코미디
- 주요 확장 분야: 아트토이, 애니메이션, 굿즈, 출판, 게임, 전시
- 기본 포지셔닝: 낯선 감정을 호기심과 놀이로 바꾸는 캐릭터

1. 새로운 감정을 발견하게 한다.
2. 신댕이의 독특한 행동을 보여준다.
3. 수집 욕구를 만든다.
