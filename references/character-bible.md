# Character Bible

## 캐릭터 DNA

캐릭터의 핵심은 외형보다 **반복 가능한 행동 패턴**이다.

### 성격 축

- 호기심 ↔ 경계심
- 장난기 ↔ 수줍음
- 독립성 ↔ 애착
- 낙관 ↔ 몽상
- 즉흥성 ↔ 관찰성

### 행동 규칙

- 모르는 것을 보면 가까이 다가가기보다 먼저 옆눈으로 본다.
- 감정이 커질수록 몸보다 머리와 소품이 먼저 반응한다.
- 기쁠 때 크게 웃기보다 입꼬리와 어깨가 먼저 올라간다.
- 실수했을 때 숨기려다가 더 큰 상황을 만든다.
- 누군가를 위로할 때 말보다 곁에 앉는다.

## 형태 규칙

- 머리 실루엣은 최우선 식별 요소
- 얼굴 중심은 낮고, 여백이 충분해야 함
- 눈 간격과 크기는 정면·측면에서도 동일한 인상을 유지
- 손과 발은 작은 디테일보다는 명확한 덩어리감
- 의상은 3D 제작 시 분할 가능한 구조
