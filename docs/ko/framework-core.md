# 프레임워크 코어

AIPDF는 캐릭터 IP 개발을 서로 연결된 여섯 개 시스템으로 구성합니다.

## Character Core
실루엣, 얼굴 랜드마크, 비율, 의상 구조, 성격, 표정, 포즈, 턴어라운드를 정의합니다.

## World & Emotion
Emotion Bloom™을 통해 감정과 제스처, 컬러, 소품, 장소, 에피소드, 제품을 연결합니다.

## Brand System
아이덴티티, 컬러, 타이포그래피, 레이아웃, 아이콘, 다이어그램, 카피 톤을 정의합니다.

## Product System
캐릭터를 생산 가능한 아트토이, 봉제, 문구, 생활소품, 패키지, 리테일, 전시로 변환합니다.

## Business System
타깃, 채널, 라이선싱 카테고리, 파트너십, 수익 모델, 가정치와 로드맵을 연결합니다.

## Publishing System
PPTX, PDF, DOCX, 문서 사이트, 자산 목록, 버전, 권리 표기를 동기화합니다.

## 필수 프로젝트 입력

- IP명과 발음
- 레퍼런스 이미지 또는 턴어라운드
- 성격 키워드 3~7개
- 타깃과 시장
- 핵심 감정
- 세계관 현실성 수준
- 우선 제품과 매체
- 발표 대상

정보가 부족할 때는 사실처럼 추측하지 말고 가정 사항을 명시합니다.
