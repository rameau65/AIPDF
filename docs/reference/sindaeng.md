# SINDAENG Reference Implementation

SINDAENG(신댕이) is the first reference IP used to test AIPDF from character identity through final publishing.

## Positioning

A curious, playful, emotionally expressive character who turns unfamiliar feelings into small adventures.

## Core Keywords

- Curious
- Playful
- Oddly Cute
- Emotional
- Collectible
- Imaginative
- Warm

## Reference Deliverables

- AAA character and turnaround sheets
- Expression and pose libraries
- Emotion Bloom™
- Brand and visual asset system
- World and relationship maps
- Art toy and packaging visualization
- Merchandise and retail concepts
- Investor PPTX, print PDF, and editable DOCX

## Reuse Rule

SINDAENG-specific visual assets and identity details are examples, not universal defaults. To apply AIPDF to another IP, replace the project profile while preserving the framework stages and quality gates.
