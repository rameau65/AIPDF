# Framework Core

AIPDF organizes character-IP development into six connected systems.

## Character Core
Defines silhouette, face landmarks, proportions, costume construction, personality, expressions, poses, and turnaround views.

## World & Emotion
Uses Emotion Bloom™ to connect emotional states with gestures, colors, props, locations, episodes, and products.

## Brand System
Defines identity, color, typography, layout, iconography, diagrams, copy tone, and cross-channel consistency.

## Product System
Translates the character into manufacturable art toys, plush, stationery, lifestyle goods, packaging, retail, and exhibitions.

## Business System
Connects audience, channels, licensing categories, partnerships, revenue models, assumptions, and roadmaps.

## Publishing System
Synchronizes PPTX, PDF, DOCX, documentation, asset manifests, versioning, and rights notices.

## Required Project Inputs

- IP name and pronunciation
- Reference images or turnaround
- Three to seven personality keywords
- Audience and market
- Core emotions
- World realism level
- Priority products and media
- Presentation audience

When information is incomplete, document assumptions instead of silently inventing facts.
