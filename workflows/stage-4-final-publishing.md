# Stage 4 — Final Publishing

## 목적
투자자, 라이선시, 유통사, 내부 제작팀이 동일한 IP를 이해하도록 최종 출판한다.

## PPTX
- 16:9
- 3840×2160 기준
- 35~50페이지
- 편집 가능한 도형 우선
- 투자 발표 흐름: 문제 → 기회 → IP → 제품 → 시장 → 비즈니스 모델 → 로드맵 → 요청사항

## PDF
- 인쇄 및 배포
- 폰트 임베딩 확인
- 이미지 300dpi 권장
- 페이지 번호 및 문서 버전 표시
- 과도한 투명도와 애니메이션 제거

## DOCX
- 발표 원고와 상세 설명
- 제목 구조 유지
- 표와 캡션 포함
- 이미지 대체텍스트 작성
- 수정 이력 기록 가능 구조

## 동기화 규칙
PPTX, PDF, DOCX의 수치, 명칭, 로드맵, 버전은 동일해야 한다.
